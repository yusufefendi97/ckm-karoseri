1. Overview

MLBB Highlight Uploader ("the Application") is a private desktop tool that allows users to publish Mobile Legends: Bang Bang gameplay highlight videos to their own TikTok accounts via TikTok's official Content Posting API. This Privacy Policy explains how we handle information in connection with the Application.

2. Information We Collect

The Application collects the minimum information necessary to function:

  • TikTok OAuth tokens — An access token and refresh token issued by TikTok after the user grants permission. These tokens are stored locally on the user's own device and are never transmitted to our servers.
  • TikTok account display name and open ID — Retrieved via the user.info.basic scope solely to identify which account is connected in the application interface.
We do NOT collect names, email addresses, passwords, payment information, location data, device identifiers, or any analytics data.

3. How We Use Information

Information is used solely to:

  • Authenticate the user's TikTok account using OAuth 2.0
  • Display the connected account name in the application
  • Upload video files selected by the user to their TikTok account

4. Data Storage

All data (OAuth tokens, account information, video queue settings) is stored exclusively on the user's local device in a configuration file. No data is stored on remote servers operated by Fend Network.

5. Data Sharing

We do not share, sell, rent, or trade any user data with third parties. The only external service the Application communicates with is TikTok's official API (open.tiktokapis.com).

6. TikTok API Data Use

This Application uses TikTok's Content Posting API. By connecting your TikTok account, you also agree to TikTok's own Terms of Service and Privacy Policy. Data received from TikTok's API is used only within the Application for the purpose of publishing content on your behalf and is never shared or stored externally.

7. Security

OAuth tokens are stored locally on your device. You are responsible for securing your own device. If you believe your account has been compromised, you can revoke the Application's access at any time through TikTok's account security settings at tiktok.com/setting under Manage app permissions.

8. Data Retention & Deletion

You may delete all stored data at any time by:

  • Removing your account from the Application (deletes stored tokens from the config file)
  • Revoking the Application's access from your TikTok account settings
  • Manually deleting the Application's configuration files from your device

9. Children's Privacy

The Application is not intended for use by individuals under the age of 13. We do not knowingly collect information from children.

We may update this Privacy Policy from time to time. The latest version will always be published at this URL. Continued use of the Application after changes constitutes acceptance of the updated policy.
