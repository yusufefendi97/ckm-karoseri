1. Overview

This Privacy Policy describes how the CKM Karoseri TikTok Content Tool ("Application") handles information. This is a private desktop application used solely by the owner of CKM Karoseri.

2. Data We Do NOT Collect

  • We do not collect personal data from any third party
  • We do not store user credentials beyond the local device
  • We do not share any data with external services other than TikTok's official API
  • We do not use cookies or tracking technologies

3. TikTok API Usage

This application uses TikTok's official Content Posting API solely to upload video content on behalf of the authenticated account owner. Access tokens are stored locally on the owner's device and are never transmitted to any third-party server.

4. Data Storage

All data (access tokens, account settings) is stored locally on the user's device. No cloud storage or external database is used.
