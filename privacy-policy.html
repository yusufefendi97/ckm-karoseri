1. Overview

This Privacy Policy explains how the CKM Karoseri TikTok Content Tool ("Application", "we", "our") handles information in connection with your use of the Application. This is a private desktop tool that connects to TikTok's official Content Posting API solely to publish video content on behalf of the authenticated account owner.

We are committed to protecting your privacy. This policy describes what information is handled, how it is used, and the rights available to you.

2. Information We Handle

The Application handles only the minimum data necessary to function:

  • TikTok OAuth Access Token & Refresh Token — Obtained after the user completes TikTok's official login flow. Stored locally on the user's device only.
  • TikTok Open ID — A unique identifier provided by TikTok to identify the authenticated account. Used only to verify the correct account is being posted to.
  • Video files — Original video files selected by the user from their local device. These are uploaded directly to TikTok via the official API and are not stored or transmitted elsewhere.
We do NOT collect names, email addresses, passwords, payment information, location data, or any behavioral/analytics data.

3. How We Use Information

All data handled by the Application is used exclusively for the following purposes:

  • Authenticating with TikTok's API on your behalf using your authorized access token
  • Verifying that content is published to the correct authorized TikTok account
  • Uploading user-selected video content to TikTok via the Content Posting API

We do not use your information for advertising, profiling, analytics, or any purpose beyond the core functionality described above.

4. Local Data Storage

All data (access tokens, account configuration) is stored locally on the user's own device in a configuration file. No cloud storage, external database, or remote server is used. The Application does not transmit any data to servers operated by CKM Karoseri.

5. Data Security

We implement the following measures to protect your data:

  • Tokens are stored in a local configuration file accessible only to the operating system user running the application
  • All communication with TikTok's API is performed over HTTPS (TLS-encrypted connections)
  • No sensitive data is logged to log files
  • The Application does not transmit credentials to any third-party service

6. Third-Party Services

The Application integrates solely with the TikTok API (operated by TikTok Technology Limited and affiliates). Data transmitted to TikTok is governed by TikTok's own Privacy Policy, available at tiktok.com/legal/privacy-policy.

We do not integrate with, sell data to, or share data with any other third parties.

7. Data Retention & Deletion

Access tokens and account configuration are retained locally on the user's device for as long as the Application is in use. You may delete this data at any time by:

  • Removing your account from the Application (which deletes the stored tokens from the configuration file)
  • Revoking the Application's access directly from your TikTok account settings at tiktok.com/setting under Manage app permissions
  • Manually deleting the Application's configuration files from your device

Upon revoking access or removing the account, all locally stored tokens are deleted and the Application can no longer interact with your TikTok account.

8. Your Rights

As a user of this Application, you have full control over your data:

  • Access: You can view all stored data by inspecting the local configuration file on your device
  • Deletion: You can delete all stored data at any time as described in Section 7
  • Revocation: You can revoke TikTok API access at any time from your TikTok account settings
  • Correction: You can re-authenticate at any time to update your account credentials

9. Children's Privacy

This Application is intended for use by adults (18 years and older) who own and operate TikTok business or creator accounts. We do not knowingly collect or process any information from individuals under the age of 13. If you believe a minor has inadvertently used this Application, please contact us immediately at the address below.

Continued use of the Application after any changes constitutes acceptance of the updated policy. We encourage you to review this policy periodically.
