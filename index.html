About This Application

MLBB Highlight Uploader is a private desktop application designed to help Mobile Legends: Bang Bang content creators and pro players automatically publish their gameplay highlight videos to TikTok.

The tool connects to TikTok's official Content Posting API using the standard OAuth 2.0 authentication flow. All publishing actions are fully authorized by the account owner. No credentials or personal data are ever transmitted to any third-party server.

How It Works

Step 1 — Connect Account: Log in via TikTok's official OAuth 2.0 flow to grant the application permission to publish on your behalf.
Step 2 — Select Highlights: Choose Mobile Legends gameplay video files from your local device to queue for publishing.
Step 3 — Schedule & Publish: Set a publish schedule or upload immediately. The application handles the rest using TikTok's Content Posting API.

API Scopes & Permissions

This application requests only the minimum permissions necessary to function:

  • user.info.basic — Verify the authenticated TikTok profile to ensure content is published to the correct account.
  • video.publish — Upload and publish gameplay highlight videos from the user's local device to their TikTok account.

No other scopes or permissions are requested. The application does not read, modify, or delete existing TikTok content.

Security & Privacy

  • All authentication is handled directly by TikTok's official OAuth 2.0 system
  • Access tokens are stored locally on the user's own device only
  • No user data, videos, or credentials are transmitted to any third-party server
  • The application does not collect analytics or usage data
  • Only original content owned by the user is published
