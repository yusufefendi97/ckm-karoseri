About This Application

CKM Karoseri Content Publisher is a private desktop application designed to help creators automate the publishing of their original video content to TikTok securely and efficiently.

The tool connects to TikTok's official Content Posting API using the standard OAuth 2.0 authentication flow, ensuring all actions are fully authorized by the account owner. No credentials or personal data are ever transmitted to any third-party server.

How It Works

Step 1 — Authenticate: The user logs in via TikTok's official OAuth 2.0 flow to grant the application permission to post on their behalf.
Step 2 — Select Content: The user selects original video files from their local device directory.
Step 3 — Publish: The application uploads the video directly to the user's TikTok account using the Content Posting API. All posts are user-initiated.

API Scopes & Permissions

This application requests only the minimum permissions necessary to function:

  • user.info.basic — Used to verify the authenticated TikTok profile and ensure content is published to the correct account.
  • video.publish — Used to upload and publish original video content from the user's local device to their TikTok account.

No other scopes or permissions are requested. The application does not read, modify, or delete existing TikTok content.

Security & Privacy

  • All authentication is handled directly by TikTok's official OAuth 2.0 system
  • Access tokens are stored locally on the user's own device only
  • No user data, videos, or credentials are transmitted to any third-party server
  • The application does not collect analytics or usage data
  • Only original content owned by the user is published
